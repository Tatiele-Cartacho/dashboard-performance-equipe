# 📈 Performance Team Analytics

### 🎯 Sobre o Projeto
Este dashboard foi desenvolvido para auxiliar na gestão de recursos humanos e acompanhamento de metas. O objetivo é fornecer uma visão clara do desempenho individual e coletivo, permitindo identificar pontos de melhoria e reconhecer resultados de alta performance dentro da organização.

### 🛠 Tecnologias e Habilidades
* **Ferramenta:** Power BI
* **Lógica e Cálculos:** DAX (Data Analysis Expressions) - *Foco em métricas de desempenho e acompanhamento de metas.*
  
### 🔍 Principais Funcionalidades
* **Acompanhamento de Metas:** Comparativo direto entre o que foi planejado vs. o que foi realizado.
* **Monitoramento de Indicadores:** Visualização clara do desempenho de cada colaborador.
* **Filtros Interativos:** Segmentação de dados para análises rápidas por período ou equipe.
